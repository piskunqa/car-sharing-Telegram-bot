Бот для поиска попутного транспорта
1. Водители создают поездки когда планируют куда-то ехать, в какую дату и сколько у них есть свободных мест
2. Те кому нужна попутка отправляют боту точку посадки и куда им нужно
3. Если часть маршрута одного или нескольких водителей совпадает с маршрутом пассажира, пассажиру предлагается забронировать место
4. Есть возможность сохранить шаблон поездки
5. Поддержка нескольких языков
